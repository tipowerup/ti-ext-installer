# TI Powerup Installer

A custom TastyIgniter extension

## Installation

```bash
composer require tipowerup/ti-ext-installer
```

Then install in TastyIgniter:

```bash
php artisan igniter:extension-install tipowerup.installer
```

## Requirements

- TastyIgniter v4.0+
- PHP 8.2+
